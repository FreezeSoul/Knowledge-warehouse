# kubenetes

## kubenetes安装部署

kubenetes 安装部署参考 github 项目 [kubeasz](https://github.com/gjmzj/kubeasz)

`kubeasz`致力于提供快速部署高可用`k8s`集群的工具, 并且也努力成为`k8s`实践、使用的参考书；基于二进制方式部署和利用`ansible-playbook`实现自动化：即提供一键安装脚本, 也可以分步执行安装各个组件, 同时讲解每一步主要参数配置和注意事项。
