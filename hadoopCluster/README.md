# README


本文档介绍的集群搭建环境基于以下平台及软件

1. GNU/Linux  CentOS 7.4
2. java 1.8.X 必须安装，建议选择Sun公司发行的Java版本。

# 背景

公司准备开始进行新闻分析项目，项目涉及到大数据环境与数据etl。
本文档用于记录项目大数据、etl基本的环境搭建。

# 文档目标

本文档主要介绍了hadoop集群及部分大数据相关程序的安装部署、状态监测、服务启停方式。
