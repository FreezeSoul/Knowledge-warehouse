# 运维工程师-进阶之路

## 初衷

创建这个知识仓库的主要目的是加强自己的知识沉淀。

作为运维在工作中会接触到各种技术，网络、数据库、python、go、容器、操作系统等等。在学习新的技术的时候，我们会用文字和图片将我们的知识记录下来，比如使用博客记录是一种很好的方式。

但是仅仅记录下来是远远不够的，我们需要更加系统的将我们的知识做管理、分类。

以下我开始知识仓库的初衷：

1. 以一种更加系统的方式将知识进行总结归类，巩固知识点。

2. 在需要使用某方面的时候，能够快速的温习。

3. 在发现知识盲区，完善自己的知识板块。
